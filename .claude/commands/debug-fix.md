Find and fix a specific bug with a minimal surgical edit.

Input: $ARGUMENTS (error message or description of what's broken)

Steps:
1. Identify which file and function is responsible for the error
2. Read the current state of that file
3. State the fix in exactly one sentence:
   - What the error is
   - What file and line it's in
   - What the fix is
4. Apply only that fix — nothing else
5. Report what was changed

Rules:
- NEVER change the overall approach
- NEVER rewrite the file
- NEVER refactor surrounding code
- Fix only what is broken
